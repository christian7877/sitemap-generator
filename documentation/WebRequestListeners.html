new WebRequestListeners( requestDomain, successCodes, contenttypePatterns, callbacks )

Description

creating this class will attach webrequest listerers to runtime allowing monitoring http request and tab load state. Once all processing is done call destroy() method to detact the listeners.

Parameters
Name Type Description
requestDomain String

base url to monitor
successCodes Array.<Number>

list of http status codes that are considered successsful
contenttypePatterns Array.<String>

list of content types that should be crawled; when content type is detected that falls outside this list, the tab will immediately close
callbacks Object

callback functions to handle various events

Name Type Description
onMessage function

when chrome.runtime.onMessage occurs
callback.onNext function

when next url should launch
callback.onUrls function

when some url has been discovered
callback.onTerminate function

when contentScript fails to load we should terminate
callback.onError function

handler when some individual url returns error response
callback.onUrlError function

handler when some individual url returns success response
